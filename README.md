# Sistema Clínica SePrise - Consultorios Externos

## Materia
Desarrollo de Sistemas de Información orientados a la gestión y apoyo a las decisiones - PP2

## Grupo
Grupo 5 - Comisión B

## Tecnología
C# Windows Forms - .NET 6.0 Windows

## Cómo ejecutar
1. Descargar o clonar el repositorio.
2. Abrir `DSI_PP2_G5_ClinicaSePrise_CE.sln` en Visual Studio 2022.
3. Verificar que esté instalado .NET 6.
4. Ejecutar el proyecto.
